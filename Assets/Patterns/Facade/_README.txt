Facade is a structural design pattern that provides a simplified interface to a library, a framework, or any other complex set of classes. The main difference between the facade and the adapter pattern is that the facade pattern does not change the interface of the system, but only simplifies it, while the adapter pattern changes the interface of the system to match another one.

Use the Facade pattern when you need to have a limited but straightforward interface to a complex subsystem.

Use the Facade when you want to structure a subsystem into layers.
